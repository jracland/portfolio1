My Story

I had my first experience developing websites using Wordpress to publish a blog which taught conversational English. However, I quickly got frustrated by not being able to personalise the templates to get them looking exactly the way I wanted them. I needed more control.

A while later, I built a centralised online database system for storing a primary school's curriculum, planning, and assessment data using G-Suite, mostly in Google Sheets. I enjoyed problem-solving and finding ways to link the documents together using snippets of code and decided I needed to learn how to program web-based applications and websites for myself.

I started learning to code in November 2021. I quickly picked up HTML and CSS and practised implementing my web design skills by building some simple personal sites and completing online challenges. Immediately, I was hooked.
